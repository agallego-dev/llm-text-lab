# llm-text-lab

Primer proyecto de aprendizaje con Python y OpenAI API.

## Funcionalidad
- Lee un archivo de texto
- Envía el contenido a un modelo LLM
- Devuelve:
  - resumen breve
  - puntos clave
  - clasificación del tema principal

## Ejecución

Activar entorno virtual y ejecutar:

```bash
python -m app.main
